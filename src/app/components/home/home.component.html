Deploy on

Getting started with OpenEBS on your Kubernetes is easy. On your kubemaster, use the following command

kubectl apply -f https://
openebs.github.io
/charts/openebs-operator.yaml

That's it ! OpenEBS is now enabled on your kubernetes cluster. You can start using OpenEBS storage by specifying the openebs-standard storage class in your application's PVCs
